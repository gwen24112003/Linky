# Design Skill

Expertise en design UI/UX pour creer des interfaces modernes, accessibles et performantes.

## Principles

### Mobile-First
- Concevoir pour mobile d'abord, puis agrandir
- Touch targets minimum 44x44px
- Navigation simple sur mobile

### Accessibilite (WCAG 2.2)
- Contraste minimum 4.5:1 pour le texte
- Navigation clavier complete
- Attributs alt sur toutes images
- Labels sur tous les formulaires

### Design System
- Coherence des couleurs et typographie
- Espaces et alignements reguliers
- Composants reutilisables

## Quick Checks

### Responsive
- [ ] Mobile: 320px - 767px
- [ ] Tablet: 768px - 1023px  
- [ ] Desktop: 1024px+

### Performance
- [ ] Images optimisees
- [ ] Animations < 300ms
- [ ] Pas de layout shift (CLS < 0.1)

### Accessibilite
- [ ] Contraste OK
- [ ] Focus visible
- [ ] ARIA si besoin
- [ ] H1 present
- [ ] Contenu substantiel
