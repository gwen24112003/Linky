# Accessibilite (WCAG 2.2)

## Contraste

- Texte normal: 4.5:1 minimum
- Grand texte (18px+ ou 14px bold): 3:1 minimum
- Composants UI: 3:1

## Navigation

- Focus visible sur tous les elements interactifs
- Ordre logique au clavier
- Skip links pour le contenu principal

## Images

- Alt text descriptif
- `alt=""` pour images decoratives
-  role="img" si necessaire

## Formulaires

- Labels associes aux champs
- Messages d'erreur explicites
- Required fields clairement indiques

## Contenu

- H1 unique par page
- Structure H1 > H2 > H3 logique
- Langue specifiee (lang="fr")
